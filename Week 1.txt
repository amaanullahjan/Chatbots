For Environment Setting:-
i-     Create an account on Github.com
ii-    Create Repository after loging on Github.com
iii-   Download git from google or other Search Engine which you use
iv-    Open cmd
v-     For cloning write command in cmd
       git clone "which URL you copy from github"
vi-    For checking git either clone or not write command 
       git -v
vii-   Download VS Code and then create Text file
       and write some text.
viii-  For adding the data write command 
       git add .      
ix-    After write some text then for commit write the command 
       git commit -m "message"     commit means for your Memory remind   
x-     For sending or pushing the text, code or data write command
       git push
